# user-api
Rest Api with Spring Boot User Authentication Authorization , Spring Security and Redis Session Management

# technologies

- Spring Boot
- Spring Security
- Redis for Sessions
- MySQL

# features

- Authentication +
- Authorization +
- Login +
- Register +
- Update User Info Without Password
- Change Password
- Forget Password *
- Confirm Email +
- Delete User Account
- Block User Account

# refoctoring

- create docker-compose file
- use email as username
- model validation
